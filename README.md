## 🎯 О проекте

**BookSwap** — сервис, позволяющий пользователям размещать свои книги и обмениваться ими с другими читателями. Основные принципы: бесплатно, без посредников, с системой рейтингов и отзывов.

### Реализованный функционал

| Функция | Статус |
|---|---|
| Регистрация и аутентификация (JWT) | ✅ |
| Каталог книг с поиском и фильтрацией | ✅ |
| Добавление / редактирование / удаление книг | ✅ |
| Предложение, принятие и завершение обмена | ✅ |
| Система отзывов и рейтингов | ✅ |
| Профиль пользователя с историей | ✅ |
| REST API с авто-документацией (Swagger) | ✅ |
| React SPA с адаптивным дизайном | ✅ |

## 🏗 Архитектура

```
Браузер (React SPA)
        │
        │  HTTP / REST JSON
        ▼
  FastAPI (Python)
  ┌─────────────────────────────┐
  │  /api/auth   — JWT-auth     │
  │  /api/books  — книги        │
  │  /api/users  — пользователи │
  │  /api/exchanges — обмены    │
  └──────────────┬──────────────┘
                 │ SQLAlchemy ORM
                 ▼
           SQLite (файл БД)
```

**Жизненный цикл обмена:**
```
pending → accepted → completed
       │         └→ cancelled
       └→ rejected
```

**Статус книги:**
```
available → in_exchange → exchanged
         ↑               (при отмене/отклонении)
         └───────────────────────────────────────
```

## 🚀 Запуск проекта

### Требования
- Python 3.11+
- Node.js 18+

### 1. Клонировать репозиторий
```bash
git clone https://github.com/<your-username>/ssrvp.git
cd ssrvp
```

### 2. Запустить бэкенд
```bash
pip install -r requirements.txt
python run.py
```
Сервер запустится на **http://localhost:8000**

### 3. Собрать фронтенд

**Production** (рекомендуется):
```bash
cd frontend
npm install
npm run build
```
После сборки фронтенд раздаётся бэкендом — открывайте **http://localhost:8000**

**Dev-режим с hot reload:**
```bash
cd frontend
npm run dev   # http://localhost:3000
```

### 4. Добавить тестовые данные
```bash
python -X utf8 seed.py
```

## 📖 API Документация

После запуска бэкенда:
- **Swagger UI**: http://localhost:8000/api/docs
- **ReDoc**: http://localhost:8000/api/redoc

### Эндпоинты

```
POST   /api/auth/register           — Регистрация
POST   /api/auth/login              — Вход, получение JWT
GET    /api/auth/me                 — Текущий пользователь

GET    /api/books/                  — Список книг (фильтры: search, genre, condition, city)
POST   /api/books/                  — Добавить книгу           [auth]
GET    /api/books/{id}              — Книга по ID
PUT    /api/books/{id}              — Редактировать книгу      [auth]
DELETE /api/books/{id}              — Удалить книгу            [auth]

GET    /api/users/{id}              — Профиль пользователя
GET    /api/users/{id}/books        — Книги пользователя
GET    /api/users/{id}/reviews      — Отзывы пользователя
PUT    /api/users/me                — Обновить свой профиль    [auth]

GET    /api/exchanges/              — Мои обмены               [auth]
POST   /api/exchanges/              — Предложить обмен         [auth]
PUT    /api/exchanges/{id}/accept   — Принять запрос           [auth]
PUT    /api/exchanges/{id}/reject   — Отклонить запрос         [auth]
PUT    /api/exchanges/{id}/complete — Завершить обмен          [auth]
PUT    /api/exchanges/{id}/cancel   — Отменить запрос          [auth]
POST   /api/exchanges/reviews       — Оставить отзыв           [auth]
```

## 📝 User Stories (INVEST)

### US-1: Регистрация и вход
> **Как** новый пользователь,  
> **я хочу** зарегистрироваться по email и войти в систему,  
> **чтобы** получить доступ к обмену книгами.

| Критерий INVEST | Описание |
|---|---|
| **I**ndependent | Не зависит от других историй |
| **N**egotiable | Можно добавить OAuth / соцсети позже |
| **V**aluable | Без входа нет доступа к ключевым функциям |
| **E**stimable | ~1 спринт |
| **S**mall | Только email + пароль, без 2FA |
| **T**estable | Форма → API → токен → редирект на главную |

**Критерии приёмки:**
- Форма с валидацией: email, username, пароль (мин. 6 символов), совпадение паролей
- JWT сохраняется в localStorage, navbar показывает аватар пользователя
- При повторном открытии сайта — сессия восстанавливается

---

### US-2: Добавление книги
> **Как** зарегистрированный пользователь,  
> **я хочу** добавить свою книгу в каталог,  
> **чтобы** другие пользователи могли предложить мне обмен.

| Критерий INVEST | Описание |
|---|---|
| **I**ndependent | Независима от обменов |
| **N**egotiable | Загрузка фото книги — за рамками MVP |
| **V**aluable | Основа всего сервиса — без книг нет обменов |
| **E**stimable | ~0.5 спринта |
| **S**mall | Одна форма: название, автор, жанр, состояние, описание |
| **T**estable | Книга появляется в «Мои книги» и в каталоге |

**Критерии приёмки:**
- Обязательные поля: название и автор; остальные опциональны
- После добавления книга сразу видна в каталоге со статусом «Доступна»
- Редактирование и удаление работают из страницы «Мои книги»

---

### US-3: Поиск книги в каталоге
> **Как** пользователь (в т.ч. незарегистрированный),  
> **я хочу** искать книги по названию, автору, жанру и городу,  
> **чтобы** быстро найти нужную книгу для обмена.

| Критерий INVEST | Описание |
|---|---|
| **I**ndependent | Не требует авторизации |
| **N**egotiable | Сортировка и ML-рекомендации — за рамками MVP |
| **V**aluable | Без поиска каталог бесполезен при большом числе книг |
| **E**stimable | ~1 спринт |
| **S**mall | Поиск по тексту + фильтры (без ElasticSearch) |
| **T**estable | Результаты соответствуют введённым фильтрам |

**Критерии приёмки:**
- Текстовый поиск по названию и автору (Enter или кнопка «Применить»)
- Фильтры по жанру, состоянию, городу работают независимо и вместе
- Пагинация по 12 книг, кнопки «Далее» / «Назад»

---

### US-4: Предложение обмена
> **Как** авторизованный пользователь,  
> **я хочу** предложить обмен владельцу понравившейся книги,  
> **чтобы** договориться о встрече и получить книгу.

| Критерий INVEST | Описание |
|---|---|
| **I**ndependent | Зависит от US-1 и US-2 |
| **N**egotiable | Обмен без предложения своей книги допустим |
| **V**aluable | Ключевой бизнес-процесс всего сервиса |
| **E**stimable | ~1–2 спринта |
| **S**mall | Один запрос за один раз, без торгов |
| **T**estable | Запрос виден у владельца во вкладке «Входящие» |

**Критерии приёмки:**
- Кнопка «Предложить обмен» видна только для чужих книг со статусом «Доступна»
- Можно выбрать свою книгу взамен или отправить просто так с сообщением
- Нельзя отправить два одинаковых запроса на одну книгу

---

### US-5: Управление обменами
> **Как** владелец книги,  
> **я хочу** принимать, отклонять и завершать обмены,  
> **чтобы** контролировать, кому отдать свои книги.

| Критерий INVEST | Описание |
|---|---|
| **I**ndependent | Зависит только от US-4 |
| **N**egotiable | Push-уведомления — за рамками MVP |
| **V**aluable | Без управления обмен невозможно завершить |
| **E**stimable | ~1 спринт |
| **S**mall | 4 действия: принять / отклонить / завершить / отменить |
| **T**estable | Статусы меняются, кнопки появляются/исчезают по ролям |

**Критерии приёмки:**
- Три вкладки с счётчиками: Входящие / Исходящие / Завершённые
- При принятии обе книги переходят в статус «В процессе»
- После «Завершить» обе книги получают статус «Обменяна»
- Инициатор может отменить запрос на любой стадии (pending/accepted)

---

### US-6: Отзывы и рейтинг
> **Как** участник завершённого обмена,  
> **я хочу** оставить отзыв с оценкой партнёру,  
> **чтобы** другие пользователи знали, насколько он надёжен.

| Критерий INVEST | Описание |
|---|---|
| **I**ndependent | Зависит от US-5 |
| **N**egotiable | Ответ на отзыв, жалобы — за рамками MVP |
| **V**aluable | Формирует доверие в сообществе читателей |
| **E**stimable | ~0.5 спринта |
| **S**mall | Оценка 1–5 + необязательный текст |
| **T**estable | Рейтинг пересчитывается, отзыв виден в профиле |

**Критерии приёмки:**
- Кнопка «Оставить отзыв» появляется только после статуса «Завершён»
- Интерактивный выбор звёзд с hover-эффектом
- Один отзыв на один обмен (повторная попытка → ошибка)
- Рейтинг профиля = среднее арифметическое всех оценок
